收集了一些前期的图像视频芯片的信息

周一 16 一月 2023

现代的计算机系统，无论是台式电脑、笔记本、手机、游戏机，基本上都有专门的图形芯片，用于图像的输出。今天，我们一般将这种图形芯片，称为GPU（Graphic Processing Unit），即图形处理单元。

除了一些作为控制器使用MCU系统，一般的通用型计算机系统，都包括了CPU。在PC上，GPU在显卡上存在。而手机等系统，GPU则作为SoC（System on a Chip）的一部分，在主芯片上存在。

在计算机的发展早期，是没有专门的芯片用于图像输出的，早期的计算机系统甚至没有图像输出。后来计算机越来越为个人所使用，图像输出变得重要起来。于是就有了专门的芯片。

本篇记录和罗列一些我收集到的前期图像视频芯片的信息，后面会做陆续更新。

早期图像视频芯片是没有统一的名称的，所以需要罗列一下：

缩写 全称 备注
CRTC Cathode Ray Tube (CRT) Controller
GDC Graph Display Controller
GSP Graphics System Processor TMS34010
GDP Graphics Display Processor
PPU Picture Processing Unit 任天堂红白机、超任
VDC Video Display Controller
VDG Video Display Generator Generator这个词和Controller差不多的意思
VDP Video Display Processor 世嘉Mark3、MD，MSX电脑
VIA Television Interface Adaptor 雅达利2600
VIC Video Interface Chip MOS公司

具体的图像视频芯片做一下罗列：

型号 厂商 备注 MDMD
6560 MOS 6502同期的视频接口芯片，就是VIC，以及6561
6567 MOS 和6566、8562、8564都属于VIC-II，C64使用
CDP1861 美国无线电公司 七十年代中期产品，常与RCA1802微处理器搭配
EF9345 意法半导体
SN76489 德州仪器 世嘉Mark3中使用了雅马哈生产的SN76489
TMS34010 德州仪器 芯片上还有CPU，主要用于街机
TMS9918 德州仪器 TI-99/4A，MSX初代
MC6845 摩托罗拉 常与Z80和6502在家用电脑中搭配使用
MC6847 摩托罗拉
V9938 雅马哈 用于MSX2电脑
V9958 雅马哈 用于MSX2+电脑
µPD7220 NEC NEC的PC-98

做这些信息的收集，主要是为了后面自制怀旧风格[家用电脑]做准备。
